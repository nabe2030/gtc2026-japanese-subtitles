# NVIDIA GTC 2026 基調講演 日本語字幕

Jensen Huang 基調講演（2026年3月16日、サンノゼ）の非公式日本語字幕です。  
Claude（Anthropic）を使って英語字幕から翻訳・生成しました。

## 対象動画
NVIDIA GTC 2026 Keynote（YouTube 公式）

## 使い方

### VLC（ローカル動画）
動画と同じフォルダにSRTを置くか、  
VLC → 字幕 → 字幕ファイルを追加

### ブラウザ（YouTube）
Chrome/Edge に [Language Reactor](https://chromewebstore.google.com/detail/language-reactor/hoombieeljmmljlkjmnheibnpciblicm) を導入して  
YouTubeの画面にSRTをドラッグ＆ドロップ

## 注意
- 非公式・個人翻訳です
- 技術用語（GPU/CUDA/AI/NVLink等）は英語のまま

書き終えたら **「Commit changes」** で保存。
